# 🏭 Factory Reallocation & Shipping Optimization System

(Final Year / Internship ML Project)

---

## 🎯 Project Goal

Build a machine learning system that:

* Predicts shipping lead time
* Simulates factory reassignment scenarios
* Recommends the best factory for a product
* Improves operational efficiency

Company Context: Nassau Candy Distributor

---

## 🧠 Core Idea (Simple Explanation)

Traditional factory assignment is fixed.

This system:

1. Predicts lead time using ML
2. Tests all factory options
3. Compares outcomes
4. Recommends the best factory

## ⚙️ Main Features Built

### ✔ Feature Engineering

* Lead Time Months
* Factory mapping
* Factory coordinates
* Region coordinates
* Distance_km (geodesic distance)
* Order Year / Month / Day

---

### ✔ ML Models Used

* Linear Regression (baseline)
* Random Forest Regressor (best model)

Final Performance:

* R² ≈ 0.56
* MAE ≈ 5.35 months

---

### ✔ Simulation Engine

For each product:

* Test all factories
* Predict lead time
* Rank factories by performance

Output:

Best factory recommendation.

---

### ✔ Streamlit Dashboard

Features:

* Product dropdown
* Run Optimization button
* Recommendation table
* Lead-time graph
* Best factory suggestion
